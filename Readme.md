# Persian-look Encoder/Decoder (Client-side)

یک وب‌اپ ساده برای تبدیل هر متن (URL/Config/Token/...) به خروجی فارسی‌نما و بازگشایی دقیق آن.

## حالت‌ها
- بدون پسورد: قابل بازگشایی توسط هر کسی که ابزار را داشته باشد (فقط Obfuscation)
- با پسورد: AES-GCM + PBKDF2 (بدون پسورد قابل بازگشایی نیست)

## اجرا
فقط فایل‌ها را باز کنید یا روی GitHub Pages دیپلوی کنید.
