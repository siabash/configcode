کدگذار/بازگشا (فارسی‌نما)

هر متن (URL/Config/Token/… با همه سمبل‌ها) → خروجی فارسی‌نما → برگشت دقیق

نکته: در حالت بدون پسورد، هر کسی با همین ابزار می‌تواند دیکد کند.
در حالت با پسورد، بدون پسورد خروجی قابل بازگشایی نیست.

چطور روی GitHub Pages بیارم بالا؟

  1. یک ریپو جدید بساز (Public)
  2. این فایل‌ها را داخلش بذار و Push کن
  3. Settings → Pages → Deploy from branch → branch: main / root
  4. آدرس سایتت ساخته میشه
